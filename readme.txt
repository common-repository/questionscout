== Description ==

## QuestionScout is an all-in-one online form & survey builder that makes forms and surveys easy to create, enjoyable to take, and simple to manage. 
This plugin allows you to import your QuestionScout forms and surveys into your WordPress pages & posts with just a single click. Using QuestionScout, you can create beautiful conversational & traditional forms and surveys, while having complete flexibility over the design, branding, white-label settings and much, much more!

> **Important:** This plugin is for embedding forms created using [QuestionScout]([https://questionscout.com/](https://www.questionscout.com/)). It does not allow you to create, edit or manage forms and responses within WordPress. To create manage your forms and embed them using the QuestionScout WordPress Plugin, you must first have an account created on [QuestionScout]([https://questionscout.com/](https://www.questionscout.com/)). Applicable [Terms of Service]([https://www.questionscout.com/terms-of-service](https://www.questionscout.com/terms-of-service)) and [Privacy Policy]([https://www.questionscout.com/privacy-policy](https://www.questionscout.com/privacy-policy)) apply.

== Installation ==  
## Installation Instructions  
**Manual Installation**  
1. Add the plugin folder to the /wp-content/plugins/ directory  
2. Activate the plugin through the 'Plugins' menu in WordPress  
3. Add your QuestionScout forms through the "Add block" button that has been added to post/pages editor aside of the Media Button  
4. Insert your QuestionScout Form URL and select from the options available
**Installation by using the WordPress Plugin Directory**  
1. Install the QuestionScout plugin from the WordPress Plugins directory  
2. Add your QuestionScout forms through the "Add block" button that has been added to post/pages editor aside of the Media Button  
3. Insert your QuestionScout Form URL and select from the options available

== FAQ ==  
## Frequently Asked Questions  
**Q**: *How can I add a QuestionScout Form to my content?*  
**A**: Click the "Add block" button, which opens a search field. Look for the "QuestionScout" block. Once added, click on your new block and then click the Gear icon to open your QuestionScout block settings. Paste the URL of the form you want to embed, and you're set!
